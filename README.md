# Plate Circus Game
This is the Final project assignment for the OOP class @ alexandria university
## Purpose of this project
Demonstrate understanding of OOP concepts and Implement various design patterns.

### Design patterns used:
  1.	Singleton
  2.	Factory
  3.	Strategy
  4.	Observer
  5.	Iterator
  6.	Flyweight
